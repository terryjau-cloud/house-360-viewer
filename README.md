# 360 租屋展示

以 360 全景照片與格局圖小地圖呈現三房租屋空間，讓訪客可以快速瀏覽客廳、廚房、臥室、衛浴、陽台與門外環境。

## Demo

[https://terryjau-cloud.github.io/house-360-viewer/](https://terryjau-cloud.github.io/house-360-viewer/)

## 功能特色

- 360 全景看房體驗
- 多房間場景切換
- 格局圖小地圖導航
- 目前所在位置高亮提示
- 手機版沉浸式操作介面
- 圖片缺少時顯示友善提示

## 使用技術

- HTML
- CSS
- JavaScript
- Pannellum 360 viewer CDN
- GitHub Pages
